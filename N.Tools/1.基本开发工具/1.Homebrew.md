# Mac下的包管理工具

```java
直接在终端中通过命令下载最新版本的App , 而且在安装好对应的App之后 , 其还会自动帮你配置好所有的环境变量
```

## 官网 

[http://brew.sh/index.html](http://brew.sh/index.html "Homebrew官网")

## 安装

```java
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## 使用

```java
eg - 安装Node.js
brew install node
```
